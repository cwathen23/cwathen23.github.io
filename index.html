Welcome to SuperWylbur

SuperWylbur is a commercially available MVS IBM mainframe product. It is a powerful and versatile software package designed by SuperWylbur Systems, Inc. to boost your organization's productivity.

Services

Boost Productivity

SuperWylbur uses resources efficiently. The system can support more than 1,000 simultaneous users in a single address space. SuperWylbur's built-in resource manager prevents any one user from consuming excessive machine resources.

Text Editing

SuperWylbur makes it easy to create, modify, store and retrieve text quickly and accurately, such as letters, manuals, computer programs or data.

Macros and Panels

The SuperWylbur macro processor can interact with the terminal in line mode or with full screen panels, issue SuperWylbur commands, evaluate expressions and assign the results to variables and execute conditional logic.

Job Entry and Retrieval

SuperWylbur's job entry and retrieval features allow you to submit jobs to your computer's batch stream and selectively retrieve job output at your terminal and/or route job output to central printers or remote sites.

Can I Use It?

SuperWylbur will run on:

The IBM z/OS, OS/390, MVS/ESA, MVS/XA and MVS/370 operating systems

Any IBM System/370, System/390 or compatible processor that will run an "SP" version of MVS

SuperWylbur may be accessed by:

Any IBM 3270 or compatible VTAM terminal

A VTAM logical unit type 1 device comparable to an IBM 3767 or an ASCII terminal connected by the Network Terminal Option (NTO)

An ASCII terminal using SuperWylbur's Milten telecommunications access method

SuperWylbur supports:

External Security Managers: RACF, ACF2 and TOP SECRET

System Managed Storage (SMS)

HSM

PANVALET (retrieval only)

LIBRARIAN (retrieval only)
